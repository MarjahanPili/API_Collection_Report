# Project Name : DMoney API Testing.
### Projecct Description: Dmoney is a digital wallet API used for practicing API testing and automation. It includes features like user creation, deposit, withdrawal, money transfer, and transaction history. The project validates these financial workflows using Postman and Newman to ensure accuracy, proper status codes, and error handling. It demonstrates end-to-end testing of a virtual payment system in a clean, automated way.

### Technology Used:
- Nodejs
- Postman
- Newman
- HTML Report Extra
